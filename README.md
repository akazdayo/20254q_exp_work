# プロダクト開発: 10年後のZEN Study

## コンセプト
- 学びへの平等なアクセス

## 担当別ワーク

### 概要・ターゲット・ビジネスモデル
- 概要:
  - 既存教科 + 学習すべき教科をAIが計算 + 学びたい分野も課外の分野も学べる
- ターゲット:
  - 中学生〜高校生
- ビジネスモデル:
  - 月額制

### 画面の構成・UI
- メイン画面:
  - AIアシスタントとのチャット窓
  - 今日の学習計画をAIがレコメンド
  - 学習進捗の視覚化
- 学習画面:
  - VR/AR/flat 切り替え
  - リアルタイムの字幕/翻訳機能
  - 理解度チェックのポップアップ

### 機能一覧
- パーソナライズされたAIアシスタント
- 今日の学習計画をAIがレコメンド
- 学習進捗の視覚化
- VR/AR/Flat 切り替え
- リアルタイムの字幕/翻訳機能
- 理解度チェックのポップアップ
- 24時間対応のAI質問フォーム

### 教材の一覧
